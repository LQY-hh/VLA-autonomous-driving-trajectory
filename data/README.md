# 测试数据说明

## 数据结构

```
data/
├── images/          # 前视摄像头测试图片
├── models/          # 模型权重文件
└── annotations/     # 标注数据（可选）
```

## 如何准备测试数据

### 1. 下载测试图片

您可以从以下公开数据集下载前视摄像头测试数据：

- **KITTI Dataset**：https://www.cvlibs.net/datasets/kitti/
- **BDD100K**：https://bdd100k.com/
- **Cityscapes**：https://www.cityscapes-dataset.com/

### 2. 数据组织

将下载的图片放置在 `data/images/` 目录中，建议按照以下结构组织：

```
data/images/
├── sequence_001/    # 第一组序列
├── sequence_002/    # 第二组序列
└── ...
```

### 3. 图片格式要求

- 支持的图片格式：JPG、PNG
- 建议分辨率：720p以上
- 图片命名：建议使用时间戳或序号命名，如 `000001.jpg`、`000002.jpg` 等

### 4. 模型权重

系统会自动下载YOLOv8的预训练权重。如果需要使用自定义模型，请将模型文件放置在 `data/models/` 目录中。

### 5. 标注数据（可选）

如果需要评估系统性能，可以准备标注数据，放置在 `data/annotations/` 目录中。标注格式建议使用COCO或KITTI格式。

## 示例数据

由于空间限制，本项目未包含示例数据。请按照上述说明准备自己的测试数据。

## 注意事项

- 确保测试数据符合道路场景，包含车辆、行人和交通标志等元素
- 建议准备多种场景的测试数据，如城市道路、高速公路、乡村道路等
- 测试数据量建议在100-200张图片，以确保系统性能评估的准确性
