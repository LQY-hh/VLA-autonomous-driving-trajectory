# 数据准备指南

## 1. 测试数据集下载

### 推荐数据集

1. **KITTI数据集**
   - 链接：http://www.cvlibs.net/datasets/kitti/
   - 包含前视摄像头数据和标注
   - 适合用于目标检测和轨迹预测

2. **BDD100K数据集**
   - 链接：https://bdd-data.berkeley.edu/
   - 包含多样化的驾驶场景
   - 提供详细的标注信息

3. **NuScenes数据集**
   - 链接：https://www.nuscenes.org/
   - 包含多摄像头数据
   - 适合复杂场景的轨迹预测

## 2. 数据组织

下载数据后，按以下结构组织：

```
data/
├── raw/              # 原始数据
│   ├── images/       # 原始图像
│   └── annotations/  # 标注文件
├── processed/        # 处理后的数据
│   ├── images/       # 预处理后的图像
│   └── labels/       # 处理后的标注
└── models/           # 模型文件
```

## 3. 数据预处理

### 图像预处理步骤
1. 调整图像大小到统一尺寸（如640x640）
2. 归一化像素值到0-1范围
3. 转换色彩空间（如RGB到BGR）

### 标注预处理步骤
1. 转换标注格式为模型所需格式
2. 过滤无效标注
3. 生成训练/验证/测试数据集

## 4. 示例数据

为了方便测试，可以使用以下方式获取示例数据：

1. 从KITTI数据集下载少量样本
2. 使用公开的示例图像
3. 拍摄自己的前视摄像头图像

## 5. 数据格式要求

### 图像格式
- 支持JPG、PNG格式
- 分辨率建议：720p以上
- 帧率：10-30fps

### 标注格式
- 目标检测标注：[class_id, x, y, width, height, confidence]
- 轨迹标注：[timestamp, x, y, heading]

## 6. 注意事项

1. 确保数据符合隐私要求
2. 保持数据的多样性，覆盖不同场景
3. 定期备份数据，避免数据丢失
